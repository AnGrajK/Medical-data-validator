# Medical-data-validator

The main purpose of this code is to **check if the medical records data is in the correct format** before using it.

It makes sure that:

1. The data is stored in a **list or tuple**.
2. **Each item inside the list is a dictionary** (a patient record).

If the format is correct, it prints **"Valid format."**
If something is wrong, it prints an **error message**.

 The code is used to **check that the medical records are organized properly so the program doesn’t break later.**
